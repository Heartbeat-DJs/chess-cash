### GSD2 Research Findings
- GSD2 = standalone agentic CLI built on Pi SDK for autonomous coding
- Solves **context rot** via fresh context per task, externalized state, wave-based parallelism
- Repo: github.com/gsd-build/gsd-2 | Docs: gsd.build
- **Decision: Don't install GSD2 — steal its patterns into DMN instead**
  - DMN already has the same philosophy but with multi-model advantage
  - DMN has better memory system (n8n + NotebookLM vs flat Markdown)
  - DMN has local hardware advantage (RTX 3090 = $0 tasks)

### Patterns Integrated into DMN Spec Kit

1. **Context Isolation Protocol** (Constitution Principle VIII)
   - Fresh minimal context per worker task
   - Only task desc + relevant files + style guide + system prompt
   - No accumulated history, no debug noise from other tasks

2. **Budget Containment** (Constitution Principle IX)
   - Per-task ceilings: $0.03 trivial / $0.08 standard / $0.20 complex
   - Daily aggregate: $5 | Weekly: $25
   - Workers killed on breach, task escalated

3. **Wave-Based Dispatch** (FR-018, wave-dispatcher.js)
   - Dependency analysis groups independent tasks into waves
   - Waves execute in parallel, complete fully before next wave
   - Prevents merge conflicts between dependent tasks

4. **Formalized State Machine** (FR-019, state-machine.js)
   - QUEUED → DISPATCHED → EXECUTING → IN_REVIEW → APPROVED|REJECTED|ESCALATED|BUDGET_EXCEEDED
   - All transitions timestamped and logged
